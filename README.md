# Power Electronics Engineering Study (MIT OCW Inspired)

This repository documents my self-study of advanced power electronics concepts from the MIT OpenCourseWare course *6.622 Power Electronics (Spring 2023)*.

## Topics which will be covered
- Switching converters
- Buck / Boost / Flyback topologies
- Inductor and capacitor design
- Loss analysis and efficiency
- Control concepts
- Practical engineering tradeoffs
